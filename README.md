**OrthoCare – Modern Orthopedic Hospital Website**
A modern, responsive, multi-page orthopedic hospital website built using HTML, CSS, Bootstrap, and JavaScript, designed to showcase real-world UI/UX design, structured layout development, and frontend development skills.
This project demonstrates my ability to create a professional healthcare website that delivers a clean, trustworthy, and user-friendly experience suitable for hospitals and medical institutions.

**Project Objective**
The main objective of this project is to:
Design a professional healthcare website for an orthopedic hospital
Practice real-world frontend development for medical institutions
Implement responsive layouts for mobile, tablet, and desktop
Create appointment booking and contact interaction features
Add interactive UI elements using JavaScript
Follow proper folder structure and clean, maintainable code practices
Focus on accessibility and patient-friendly navigation

**Technologies Used
HTML5**
Semantic tags for accessibility and SEO
Structured multi-page layout (Home, About, Services, Doctors, Appointment, Contact)
Clean and organized markup

** CSS3**
Flexbox & Grid for structured layout design
Custom color variables for consistent medical theme
Smooth animations and hover effects
Media queries for full responsiveness
Reusable components (cards, buttons, forms)

**Bootstrap 5**
Responsive grid system
Navbar and carousel components
Pre-built utility classes for fast UI development
Form styling and layout management

**JavaScript**
DOM manipulation
Form validation for appointment booking
Event handling (menu toggle, scroll effects, modal popups)
Dynamic UI updates (tabs, service filtering, doctor profiles)

**Libraries & Tools**
Font Awesome – Medical and UI icons
Google Fonts – Professional typography
VS Code – Development environment
Git & GitHub – Version control and project hosting

**Key Features**
Clean and modern medical UI design
Fully responsive across all devices
Online appointment booking form
Doctor profile section with specialization details
Services section (Joint Replacement, Spine Surgery, Sports Injury, Trauma Care, etc.)
Testimonials section

Contact form with Google Maps integration

Smooth scrolling navigation
